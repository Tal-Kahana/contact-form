# טופס יצירת קשר

טופס יצירת קשר בעברית עם אימות שדות ועיצוב נקי, בנוי עם HTML, CSS ו-JavaScript טהורים — ללא תלויות חיצוניות.

## איך להריץ

1. פתח את התיקייה ב-VS Code
2. לחץ על `Go Live` בשורת הסטטוס (תוסף Live Server)
3. הדפדפן ייפתח אוטומטית עם הטופס

או פשוט לחץ פעמיים על `index.html` כדי לפתוח אותו בדפדפן.

## אימותים

| שדה | כלל |
|-----|-----|
| שם מלא | לא ריק |
| אימייל | חייב להכיל `@` עם תוכן משני הצדדים |
| טלפון | 9–10 ספרות בלבד (מקפים ורווחים מותרים) |
| הודעה | לפחות 10 תווים |

- שגיאות מוצגות באדום מתחת לכל שדה
- אימות מתרחש בזמן אמת תוך כדי הקלדה
- שדה תקין מקבל מסגרת ירוקה, שגוי — אדומה

## מה למדתי

- **אימות טפסים ב-JavaScript** — כתיבת פונקציות ולידציה וחיבורן לאירועי `input` ו-`submit`
- **מניפולציה של DOM** — הוספה והסרה של classes דינמית לפי מצב השדה
- **CSS Transitions לעומת Animations** — הבנה מדוע `@keyframes` לא עובד על אלמנטים עם `display: none`, ושימוש ב-`opacity` ו-`max-height` במקום
- **Double requestAnimationFrame** — טכניקה שמבטיחה שהדפדפן יצייר את המצב ההתחלתי לפני שה-transition מתחיל
- **RTL ועיצוב בעברית** — שימוש ב-`dir="rtl"` ו-`lang="he"` לתמיכה נכונה בכיוון הטקסט
